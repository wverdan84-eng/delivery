LOJA PWA COM GOOGLE SHEETS

1. Faça uma cópia da planilha modelo
2. Publique como CSV
3. Cole o ID da planilha no index.html
4. Altere o número do WhatsApp
5. Suba os arquivos em qualquer hospedagem
6. A loja já estará funcionando

Funcionalidades:
- Loja automática
- Carrinho inteligente
- Checkout WhatsApp
- PWA instalável
- Funciona offline
- Sem mensalidade
